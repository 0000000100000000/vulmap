### 一 漏洞名
Apache dubbo 未授权访问与弱口令

### 二 漏洞描述
Dubbo是阿里巴巴公司开源的一个高性能优秀的 服务框架，使得应用可通过高性能的 RPC 实现服务的输 出和输入功能，可以和 Spring框架无缝集成。  
dubbo 因配置不当导致未授权访问漏洞。攻击者可利用此漏洞获取敏感信息，甚至控制服务器权限。
